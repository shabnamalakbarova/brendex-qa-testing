# 🛒 Brendex QA Testing Project

## 📌 Project Overview
This project represents my QA work on a real startup e-commerce platform.

---

## 🔍 Testing Scope

- Functional Testing  
- UI Testing  
- API Testing (Postman)  
- Bug Reporting  

---

## 🧪 Tested Features

- Authentication (Login/Register)  
- Product listing & categories  
- Cart functionality  
- Checkout & payment  
- Bonus system  

---

## 🐞 Key Bugs Identified

- Broken navigation leading to 404 pages  
- Payment process without card input (Critical)  
- Bonus system UI active but not functional  
- Invalid email accepted in subscription field  
- Server errors (502)  

---

## 📊 Testing Approach

- Created test scenarios based on real user flows  
- Performed exploratory testing  
- Validated API responses  
- Used Chrome DevTools for debugging
